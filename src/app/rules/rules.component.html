Правила

Математический онлайн-бой - одиночная или командная игра. В команде может участвовать не более 5 участников. Победитель определяется по следующим критериям: наибольшее количество набранных баллов, при равных баллах наименьшее количество купленных задач.

В процессе игры вы можете покупать задачи и решать их.

Для открытия задачи должно быть выполнено несколько условий:

  1. Турнир должен начаться и еще не закончиться
  2. В работе не должно быть меньше трех задач
  3. Остались еще не открытые задачи

Команда может решать только задачи, которые находятся у нее в работе. По любой из этих задач можно ввести ответ, если ответ верный, то команде начисляется указанное количество баллов и задача считается решенной. Если ответ неверный, то стоимость задачи уменьшается, после трех неверных ответов, задача считается нерешенной.

За правильное решение задачи без ошибок команда получает ее двойную стоимость. Если была одна ошибка, то команда получает изначальную стоимость задачи. При двух ошибках команда может получить лишь половину изначальной стоймости задачи.

Категорически запрещается:

  1. Списывать или подглядывать решение у соперников
  2. Пользоватся ресурсами: Wolfram|Alpha...
  3. Пытаться взломать данный сайт
